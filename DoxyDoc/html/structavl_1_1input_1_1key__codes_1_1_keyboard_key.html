enum avl::input::key_codes::KeyboardKey::KeyboardKeyCodes

Identifiers for all of the standard keyboard keys.

Note:
Only the unshifted versions of keys are represented. If you want to detect the activated of a shifted key (such as asterisk, double quote, or a curly brace), then you need to check for one of the shift modifier keys.
See also:
KeyboardModifier::KeyboardModifierCodes
