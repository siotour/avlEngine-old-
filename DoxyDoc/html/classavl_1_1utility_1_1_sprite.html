Detailed Description

Encapsulates a basic textured quadrilateral in 3-D space, oriented to be parallel to the XY plane.

Includes accessors and mutators for the spatial coordinates as well as the texture coordinates. Additionally, defines a TextureHandle type which is used to represent textures.
