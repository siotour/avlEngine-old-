Attempts to load image data from a file in the .TGA image file format.

Precondition:
file_name is the name of a color image file (not color-mapped) in .TGA format.
Postcondition:
If true is returned, then width, height, pixel_depth, contains_alpha, and pixel_data will contain information about the file file_name; you are responsible for deleting pixel_data when you no longer need the image data.
If false is returned, then there was a problem when reading from the file file_name, the file doesn't exist, or the file was improperly formatted; if this is the case, then the only gaurantees are that file_name is unchanged and pixel_data does not need to be deleted.
Parameters:
file_name[IN] Name of the TGA file to load the image data from.
width[OUT] Width of the image.
height[OUT] Height of the image.
pixel_depth[OUT] Number of bytes per pixel.
contains_alpha_channel[OUT]Whether or not the image has an alpha channel.
pixel_data[OUT] Pointer to the image's pixel data.
