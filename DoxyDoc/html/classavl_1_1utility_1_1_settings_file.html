Parses a single line of text data within lines.Determines whether or not the line of text data in lines, beginning at index line_start and ending at index line_end, is a comment, blank line, or a substantial line.

If it's a substantial line, then we will attempt to parse it into a variable name/value pair.

Precondition:
The specified line of text obeys the syntactical rules laid out in avl::utility::SettingsFile.
Parameters:
linesA set of lines of text data to be parsed into variable name/value pairs.
line_startThe index into lines of the first character on the line.
line_endThe index into lines of the last character on the line.
file_nameThe name of the settings file.
line_numberThe number of the line currently being parsed. Used to report which line syntax errors occur on.
Exceptions:
avl::utility::SyntaxErrorIf there is a syntactical error in the specified line of text data.
