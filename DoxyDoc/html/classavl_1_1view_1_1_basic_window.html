static LRESULT CALLBACK avl::view::BasicWindow::RouteWindowMessage ( HWND  window_handle,
UINT  msg,
WPARAM  w_param,
LPARAM  l_param 
) [static]

The window procedure for windows of this class.

With a few exceptions, all messages are simply fed to the default window procedure. The exceptions are:

  • WM_CLOSE is posted to the window's message queue.
  • WM_DESTROY results in calling PostQuitMessage(0).
  • WM_ACTIVATE is first posted to the window's message queue, and then fed to the default window procedure.
    Parameters:
    window_handleThe handle of the window to which the message is addressed.
    msgThe message type.
    w_paramMessage data.
    l_paramMessage data.
    Returns:
    0 if the message was handled correctly, and false if not.
