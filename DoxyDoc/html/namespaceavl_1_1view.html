bool avl::view::CheckDeviceCapabilities ( IDirect3D9 *const  d3d)

Checks to see whether or not the HAL Direct3D device on the default adapter supports all of the functionality required by the D3DRendererBase.

The functionality checked is:

  • Alpha testing is supported.
  • DrawIndexedPrimitive() is supported.
  • Hardware transformations and lightning are supported.
  • Textures with alpha channels are supported.
  • Non-power-of-two textures are supported.
  • Non-square textures are supported.
    Returns:
    True if the HAL device supports the required functionality, and false if not.
    Exceptions:
    D3DErrorIf unable to create the D3D object.
