00030                 // Behavior flags for how the text box should react to
00031                 // overstepping the maximum number of possible characters in the box.
00032                 // Behaviors are as follows:
00033                 //
00034                 //      NO_OVERWRITE - If the user reaches the maximum number of characters, any
00035                 //                                 new characters will be ignored.
00036                 //      OVERWRITE_ROWS - If the user reaches the maximum number of characters, any
00037                 //                                       additional characters will, if necessary, remove the top
00038                 //                                       row of text and scoot all rows up one position, and then
00039                 //                                       begin a new row at the bottom.
00040                 //      AUTO_ADJUST_SIZE - If the user reaches the maximum number of characters,
00041                 //                                         any additional characters will force the text box to
00042                 //                                         resize itself to accomodate the new characters. Note
00043                 //                                         that, with this behavior, the chars_per_row and rows
00044                 //                                         parameters are ignored in the constructor.
00045                 //
00046                 // Note: When using any of these behaviors, if the user attempts to display more
00047                 // characters in a textbox than is possible, all of the original characters and
00048                 // the characters attempted to be displayed by the user after the maximum size
00049                 // is reached will be stored and, in the event of the user specifying new
00050                 // dimensions for the text box, it will dynamically change how much of the text
00051                 // it displays to match the new dimensions.
00052                 enum OverwriteBehavior{NO_OVERWRITE, OVERWRITE_CHARS, OVERWRITE_ROWS, AUTO_ADJUST_SIZE};
