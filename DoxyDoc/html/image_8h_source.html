00021         // This function attempts to load image data from a file in the .TGA image file format. If an error occurs while
00022         // reading from the file, if the file doesn't exist, or if the file isn't properly formatted, false is returned
00023         // and no guarantees are made about any of the parameters except that pixel_data won't point to a new chunk of
00024         // memory (in other words, you don't need to delete pixel_data if this function fails). If the function succeeds,
00025         // true will be returned and width, height, pixel_depth, contains_alpha_channel, and pixel_data will contain
00026         // the image's data. You are responsible for delete[]ing pixel_data when you're done with it.
00027         // Note that only color images without a color map are supported.
00028         // Parameters:
00029         //              [IN]    file_name - name of the TGA file to load the image data from.
00030         //              [OUT]   width - width of the image.
00031         //              [OUT]   height - height of the image.
00032         //              [OUT]   pixel_depth - number of bytes per pixel.
00033         //              [OUT]   contains_alpha_channel - whether or not the image has an alpha channel.
