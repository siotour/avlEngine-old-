00006  * Description: Provides a partially implemented interface for a basic Direct3D renderer.
00007  * Implements the initialization of a Direct3D device and is meant to be used as a skeleton
00008  * on which to build customized Direct3D renderers. Implemented behavior is for a 32-bit
00009  * pixel format, but this can be customized by overloading the
00010  * avl::view::D3DRendererBase::InitilializeDevice() function. As long as
00011  * avl::view::D3DRendererBase::present_parameters is assigned the proper format, then
00012  * avl::D3DRendererBase::ResetDevice() should work fine. Additionally implements various
00013  * wrapper functions over Direct3D, such as for creating vertex buffers.
00014  **********/
